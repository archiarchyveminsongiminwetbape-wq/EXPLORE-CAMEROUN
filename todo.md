# Explore Afrique - Site de Tourisme Cameroun

## Structure du projet
1. **Pages principales**:
   - Page d'accueil (Index.tsx) - Présentation du site avec CTA
   - J'organise mon voyage (OrganizeTrip.tsx) - Système de réservation avec panier
   - Nos Destinations (Destinations.tsx) - Liste des destinations avec bouton "Découvrir"
   - À propos (About.tsx) - But du site
   - Blog (Blog.tsx) - Expériences utilisateurs et photos

2. **Composants à créer**:
   - Header.tsx - Navigation principale
   - Footer.tsx - Contacts et informations
   - DestinationCard.tsx - Carte pour chaque destination
   - BookingForm.tsx - Formulaire de réservation
   - ShoppingCart.tsx - Panier d'achat
   - PriceCalculator.tsx - Calculateur automatique de prix

3. **Fonctionnalités**:
   - Système de panier avec calcul automatique
   - Sélection de destinations, hébergements, restaurants
   - Réservation billets d'avion et assurance sanitaire
   - Moyens de paiement
   - Contacts WhatsApp et email

4. **Images à utiliser**:
   - Logo: final_quest_240x240__.png
   - Images touristiques: toutes les images fournies des destinations camerounaises

## Technologies utilisées
- React + TypeScript
- Shadcn-ui pour les composants
- Tailwind CSS pour le styling
- React Router pour la navigation
