# Project Summary
Explore Afrique is a tourism platform dedicated to enhancing travel experiences in Cameroon. It serves both local and international travelers by providing a comprehensive guide to destinations, accommodations, restaurants, and transport options. The website facilitates trip planning through tools for booking flights, health insurance, and managing travel needs via a shopping cart feature, ensuring a smooth and enjoyable travel experience.

# Project Module Description
- **Homepage**: Showcases the site's offerings with a clear call to action.
- **Organize My Trip**: Enables users to customize their travel plans, including selecting destinations, accommodations, and transportation.
- **Destinations**: Provides detailed information on available tourist destinations.
- **About**: Outlines the mission and values of Explore Afrique.
- **Blog**: Offers user experiences and travel tips.

# File Description Inventory

# Technology Stack
- **Frontend**: React, TypeScript
- **Styling**: Tailwind CSS, Shadcn-ui
- **Routing**: React Router
- **State Management**: React Query (for data fetching)
- **Build Tool**: Vite
